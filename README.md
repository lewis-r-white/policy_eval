# policy_eval

This repository was created to document labs and assignments for my environmental policy evaluation course. In the course, we examined the following topics:

1. Review of Multiple Regression, Heteroskedasticity, and Hypothesis Testing
2. Potential Outcomes and Fundamental Problem of Causal Inference
3. Analysis of Experimental Data
4. Regression and Matching 
5. Panel Data and Difference-in-Differences 

Assignments and in class activites are included in this repository.
